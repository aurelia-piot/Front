La boucle Do.....while

cette boucle ressemble fortement à la boucle while sauf que dans ce cas la boucle est toujours executée au moin une fois
Dans le cas d'une boucle while, si la condition n'est pas valide, la boucle n'est pas executée
Avec une do while, la boucle est executée une premirère fois, puis la condition est testée pour savoir si la condition doit continuer

Structure de do while 
        do{
            instruction 1;
            instruction 2;
            instruction 3;
            ect ...;

        }
        while(condition)
On voit bien la différence avec la boucle while, l'utilisation de la boucle do while est tres peu fréquente;

La boucle for ...in

qui permet de parcourir un objet 

            frigo = {

                fruits:["citrons","oranges"],
                legumes:["patates","courgettes"]

            };

            for(type in frigo){
                parcourt chaque clef de l'objet. type aura fruits et legume comme valeurs.
                document.write(frigo[type]);
            }
