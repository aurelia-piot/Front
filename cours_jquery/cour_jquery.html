Sélectionner en Jquery

le coeur de Jquery se trouve dans la selection d'elements pour ensuite executer des methodes dessus.

$(selecteur) accepte en argument tout les selecteur Css, Jquer nous renvoie ensuite une collection d'un ou plusieurs elements DOM enveloppés dans un obket jQuery

$("div") selectionne toute les divs
$(".titre") selectionne tous les elements qui ont la class='titre'
$("#boite") selectionne l'elements qui a id="boite"

cette enveloppe Jquery est celle qui donne accés à toutes les fonctionnalités de Jquery

  • rajoute des classes
  • modifie le contenu HTML
  • afficher ou cacher des elements
  • les animer
  • etc...

tous les elements selectionnés en javascript pur (document .getElementById,querySelector etc) sont des elements DOM et tous ceux sélectionnés avec $('...' ) sont des elements DOM dans une enveloppe Jquery

Jquery dispose de plusieurs methodes pour parcourire l'abre du document (le DOM)

  • children() parcourt les enfants directs
  • find() parcourt les descendants
  • next() renvoie le voisin suivant
  • prev() renvoie le voisin precedent
  • parent() renvoie le parent
  • eq() renvoie parmi une liste d'elements celui qui se trouve à l'index fourni (indiquer)

Les getters et les setters , ajouter et retirer des classes et savoirs si on a déjàs des classes

  • addClass(): ajoute une class
  • removeClass() : retire une class
  • toggleClass() : interrupteur ajoute/retire une class
  • hasClass() : tester l'existance d'une class
  • is() : tester si on a plusieurs class

Les evenements

la plupart des interaction de l'utilisateur avec la page web peuvent etre interceptée et une action peut etre lancée en retour

on peut dire a une fonction de s'executer quand l'utilisateur clic sur un bouton ou bien quand il scrolls la fenetre ou bienencor quand il appuis sur une touche du clavier

  • mouse(souris): click : au clic sur l'evenement
  • moseenter : lorsque la sourie passe par dessus la zonne de l'element
  • mouseleave : lorsque la souris sort de cette zone
  • keyboard (le clavier)
    keydown : lorsqu'une touche de clavier est enfoncée
    keyup : lorsqu'une touche de clavier est relachée
  • window(la fenetre du navigateur)
    scroll : defilement de la fenêtre
    resize : redimensionnement de la fenêtre
  • form(formulaires)
    change : charge les elements <input> ,<select> et <textarea> , quand l'utilisateur change une de leurs valeurs
    focus et blur : lorsque la souris est dans l'input ou sort de l'input
    submit : à l'envoi du formulaire
  • document (le contenu DOM)
    DOMcontentLoaded : lancé lorsque le document HTML est complétement chargé et analysé sans attendre que les images et le Css soient aussi chargés.

exemple
$('<p>').append('bonjour '+ $('#prenom').val()).appendTo(".info"); // $('<p>') pour creer une balise "p" avec/append('string...').appendto/dans(".info")
