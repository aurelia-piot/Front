Jquery est une bibliotheque Javascript, un Framework, qui est une boite a outils de blocs de codes réutilisable. Cette librairie est rapide petite et donc legere mais riche en fonctionnalitées. elle simplifie considérablement la navigation et la manipulatin dans le document Html (DOM) , La gestion des evenement, l'animation et Ajax(asynchronous Javascript and XML) Grace à une API (application programmin interface) facil à utiliser et qui fonctione sur une multitude de navigateur et qui est pré-tester pour nous.

Jquery est elle-meme codée en Javascript.

Polyvalence et extensibilité. JQuery a changée la façon de travailler de millions de developpeurs.

  • Fonctions toutes Faites, Optimisées, qu'on utilisera souvent: appel Ajax, boucle, animation , toggle
  • résoud beaucoup de problemes de compatiblité navigateurs. Jquery a des ployfills intégrés
  • une lecture de code plus simple car les problemes sont gérer par la librairie

Attention Jquery n'est pas à appliquer en pensement sur un code sale (tag Soup...)
